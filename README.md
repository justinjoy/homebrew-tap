# Justinjoy Tap

## How do I install these formulae?

`brew install justinjoy/tap/<formula>`

Or `brew tap justinjoy/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "justinjoy/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
